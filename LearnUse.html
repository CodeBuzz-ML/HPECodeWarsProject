Main Page


This is the main page. It allows access to all of the other components contained within this site. Get Started button, will bring you to the dashboard of the site, which can alternatively be accessed from the Navigation Bar. The other buttons in the Navigation Bar will open the respective pages.
This is the Dashboard on the home page.
  • The first option will, Open a page which contains the best tutorials available on youtube. The tutorials don't belong to EmpowerED and hence the credits to "Physics Wallah Foundation" youtube channel.

  • The second option opens a page which consists of some quizzes created by EmpowerED. A note to make, the Quizzes and the Application hosting is solely created by EmpowerED

  • The third option will open a page which contains the NCERT Textbooks. The textbooks are not republished, just hyperlinked. The TBs are directly downloaded from ncert.nic.in/textbook.php

  • The fourth option, is linked to a page. Which consists of NCERT Solutions, provided by learncbse.in and hence the credits.

  • The fifth option, opens a page consisting of some worksheets Hosted and Created exclusively by and for EmpowerED
